Propósito del proyecto

La hoja de trabajo  8 incluye la implementación de tres estructuras de almacenamiento de datos, aunque son parecidas, cada una tiene una aplicación diferente que funciona para cosas diferentes. 
El principal objetivo de este proyecto es aplicar los conceptos de simulación de estados al aplicarlos con diferentes tipos de árboles, específicamente Árbol Binario de Búsqueda, Splay Tree y Red and Black Tree, para comparar la eficiencia en la ejecución y la cantidad de iteraciones en cada uno cuando se busca, inserta o accede a procesos.
